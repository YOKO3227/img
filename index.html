사용자 가이드 Wiki

1. 기초 URL 설정

이미지가 저장된 서버의 주소를 입력하세요. 변수가 들어갈 자리는 {변수명} 형태로 작성해야 합니다.
예1: https://api.test.com/products/{v1}/{v2}.png
예2: https://api.test.com/products/{v1}{v2}.png

2. 변수 데이터 및 범위 지정

콤마(,)를 통해 개별 값을 입력하거나 대시(-)를 사용해 범위를 지정할 수 있습니다.

  • 숫자 범위: 01-10 입력 시 01, 02, ..., 10 생성
  • 단일 문자 범위: A-C 입력 시 A, B, C 생성
  • 복수 문자 범위: AA-AC 입력 시 AA, AB, AC 생성
  • 혼합 범위: A1-A3 입력 시 A1, A2, A3 생성

3. 조건부 제한 (IF 규칙)

특정 상황에서만 값이 나타나거나 사라지게 할 수 있습니다.

  • 제외: 특정 조건일 때 대상 값을 목록에서 뺍니다.
  • 만 선택: 특정 조건일 때 오직 그 값들만 노출합니다.

4. 뷰 모드 활용

생성된 결과는 두 가지 방식으로 확인 가능합니다.

  • 트리 구조 뷰: 폴더별로 이미지를 계층적으로 탐색할 때 유용합니다.
  • 갤러리 뷰: 모든 조합의 이미지를 한눈에 바둑판식으로 볼 때 유용합니다.
